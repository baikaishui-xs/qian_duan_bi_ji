# 属性
  | 参数           | 说明                       | 类型                      | 默认值 |
  | -------------- | -------------------------- | ------------------------- | ------ |
  | value          | 显示状态【值：（布尔值）】 | -                         | -      |
  | active-value   | 开启时的值                 | boolean / string / number | true   |
  | inactive-value | 关闭时的值                 | boolean / string / number | false  |

# 一、基础用法
  ```html
  <el-switch value="1" active-value="1" inactive-value="0"></el-switch>
  ```
