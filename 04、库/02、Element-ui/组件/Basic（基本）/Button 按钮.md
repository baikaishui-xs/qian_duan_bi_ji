# Button 属性
  | 属性     | 说明               | 类型               | 可选值                                             | 默认值 |
  | -------- | ------------------ | ------------------ | -------------------------------------------------- | ------ |
  | type     | 类型               | string             | primary / success / warning / danger / info / text | -      |
  | icon     | 图标               | string / Component | -                                                  | -      |
  | disabled | 按钮是否为禁用状态 | boolean            | -                                                  | -      |
  | plain    | 是否为朴素按钮     | boolean            | -                                                  | -      |
