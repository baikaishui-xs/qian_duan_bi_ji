# DOCTYPE 和 lang

**`<!DOCTYPE>` 作用：** 文档类型声明。告诉浏览器使用哪种HTML版本来显示网页

**`<!DOCTYPE html>` 作用：** 采取 HTML5 版本来显示网页

**lang 作用：** 告诉浏览器当前文档使用的语言
- en：英语
- zh-CN：中文

```html
 <!DOCTYPE html>
<html lang="zh-CN">
<head>
	<meta charset="UTF-8">
	<title>Document</title>
</head>
<body>
	你好吗 
</body>
</html>
```
