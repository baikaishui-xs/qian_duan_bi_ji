# 知识点：CSS三大特性

## 一、层叠性

**作用：** 解决样式冲突的问题

**特性：** 样式冲突时，后面的样式会层叠前面的样式

**例：**
```css
.cdx {
    /* 不生效 */
    color: red;
}
.cdx {
    /* 生效 */
    color: pink;
}
```

## 二、继承性

**作用：** 简化代码

**特性：** 
1. 子元素继承父元素的一些样式
2. 只有部分属性可以被继承。如：`text-`，`font-`，`line-` 开头的 和 `color`
3. 有些标签不能被继承。如：`<a></a>`


## 三、优先级

**特性：** 当同一个元素指定多个选择器，并且样式冲突时，就会有优先级的产生。只有权重高的选择器才会生效

**权重图：**
```
    选择器                    权重
    1、继承、*               0,0,0,0
    2、元素选择器             0,0,0,1
    3、类选择器、伪类选择器     0,0,1,0
    4、ID选择器               0,1,0,0
    5、行内样式                1,0,0,0
    6、!important             无穷大
```

**注意：** 权重并不会近位。如：0,0,0,10

**例：**
```css
#yxj {
   /* 生效 */
   color: rebeccapurple;
   /* 生效 */
   background-color: red;
}

.yxj {
   /* 不生效 */
   color: red;
   /* 生效 */
   font-size: 20px;
   /* 不生效 */
   background-color: pink!important;
}
```
