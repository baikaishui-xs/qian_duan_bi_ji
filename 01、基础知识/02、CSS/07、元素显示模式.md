# 知识点：元素显示模式

**作用：** 布局页面

## 一、块元素

**代表标签：** `<div>、...`

**特性：** 
1. 独占一行
2. 可以设置 高宽、内外边距
3. 宽度默认是父级的宽度
4. 可以存放任何元素

**语法：** `display: block`

## 二、行内元素

**代表标签：** `<span>、<a>、...`

**特性：** 
1. 一行可以显示多个
2. 不可以设置 高宽、内外边距
3. 宽度默认是内容的宽度
4. 只能存放文本和行内元素

**语法：** `display:inline`

## 三、行内块元素

**代表标签：** `<img/>、<input/>、<td>、...`

**特性：**
1. 一行可以显示多个，但是之间会有空白间隙
2. 可以设置 高宽、内外边距
3. 宽度默认是内容的宽度

**语法：** `display: inline-block`
