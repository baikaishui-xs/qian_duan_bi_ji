# 概念
  用于构建用户界面的 JavaScript 库。为什么是库不是框架？因为框架首先给拥有一整套解决方案，而 react 只是解决了视图层的问题，只有使用 react 全家桶才能算框架

# 全家桶（react 框架）
  1、react
  2、react-dom：将虚拟 DOM 渲染到 Web 应用中
  3、react-native：将虚拟 DOM 渲染到安卓和 ios 应用中
  4、react-router
  5、redux

  ## 其它
  1、react-native：安卓 和 ios 的原生应用
  2、react360：VR（虚拟现实）
